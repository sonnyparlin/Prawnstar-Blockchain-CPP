◆ handleBlockchain()

void Node::handleBlockchain ( const std::string &  blockchainString) const
Parameters
blockchainString

This is where we read and rebuild the blockchain or partial blockchain after requesting blocks from the master server. How many blocks we ad is based on how many we requested.

Ensure the incoming block count is greater than the local block count. We can do this even when downloading a partial chain because we are using the blockCount variable of the block itself rather than blocks.size().
